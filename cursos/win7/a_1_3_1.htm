Unidad 1. Avanzado: Actualización de Windows Vista a Windows 7


A pesar de que la instalación de Windows 7 se ha mejorado, logrando que sea menos interactiva y por lo tanto, más independiente en el proceso, puede ser que nos interese pasar de Windows Vista a Windows 7 simplemente instalando la actualización. Este tipo de instalación nos permite conservar los datos y las configuraciones que teníamos en el sistema, puesto que no formatearíamos el equipo.

La actualización de Windows Vista a Windows 7 sólo se puede realizar entre determinadas versiones, que son las siguientes:

     
Windows 7
      Home Premium Professional Ultimate
      32-bit 64-bit 32-bit 64-bit 32-bit 64-bit
  Home Basic 32-bit SI NO NO NO SI NO
    64-bit NO SI NO NO NO SI
  Home Premium 32-bit SI NO NO NO SI NO
Windows Vista   64-bit NO SI NO NO NO SI
  Business 32-bit NO NO SI NO SI NO
    64-bit NO NO NO SI NO SI
  Ultimate 32-bit NO NO NO NO SI NO
    64-bit NO NO NO NO NO SI

Para el resto de casos hay que instalar Windows 7 desde cero.
